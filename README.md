# Getting started

## Clone this repository 🛒
```git clone https://github.com/charisTheo/pwa-recipes.git```

## Navigate into the project's root directory 🍽
```cd pwa-recipes```

## Run the web app 🍲

### From the root directory of the project run:
```npm i && npm start```
